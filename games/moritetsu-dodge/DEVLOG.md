# 守り鉄君 セーフティ・ダッシュ 制作記録

## 企画

鉄道の安全を守るキャラクター「守り鉄君」を操作し、上から落ちてくる保守部品や工具を回避するブラウザゲーム。PCとスマートフォンの両方で遊べること、画像生成したキャラクターと落下物をゲーム素材として使用することを目標にした。

## 操作

- PCキーボード: 左右キーまたはA・Dキーで移動、スペース・上キー・Wキーで可変ジャンプ
- PCマウス: 画面下のLEFT・RIGHT・JUMPボタン
- スマートフォン: 同じ操作ボタンをタップ
- ゲーム画面: 左右端を押し続けて移動、下中央または中央をタップしてジャンプ

## 落下物

- ボルト: 小さく速い
- ナット: 小さく回転が速い
- レンチ: 細長い
- ペンチ: 変則的に回転する
- 碍子: 大きく重い
- 鋼材: 最も大きく遅い。回避得点が高い

落下物が地面に当たると短い衝撃波が発生する。左右移動だけでなくジャンプにも意味を持たせるため、地上にいる状態で衝撃波に触れるとダメージを受け、空中にいれば回避できる設計にした。

## 画像素材

添付された守り鉄君の画像とキャラクター制作指示書を参照し、ゲーム用の全身キャラクターを生成した。落下物も同じ太い輪郭線のアニメ調で個別に生成。単色背景をローカル処理で透過し、Canvasで扱いやすい最大512pxのPNGに整えた。

## 技術構成

- HTML / CSS / JavaScriptのみ
- Canvas内部解像度: 720×820
- Pointer Eventsでマウスとタッチを共通化
- Web Audio APIで効果音を生成
- ハイスコアは`localStorage`へ保存
- 14秒ごとにレベルが上がり、落下速度と発生頻度が増加
- ジャンプ入力は最大0.24秒まで保持時間を反映し、短押しと長押しで高さが変化
- キャラクター画像より左右27px内側に当たり判定を設定
- SOUNDを有効にすると簡易BGMと、落下物の重量に応じた着地音を再生
- 背景に架線柱、吊架線、トロリ線、碍子、遠景線路、定期的に通過する列車をCanvas描画
- 90秒周期で昼、夕暮れ、夜、明け方を循環し、空、山、線路、設備の色と照明を変化
- 夜間は守り鉄君のヘルメットに前照灯を描画し、光芒と足元の照明範囲を合成
- 315系、383系、385系、211系、313系を形式別デザインで描き、編成両数と速度をランダム化
- 383系は特急として高速、385系は試運転、211系は引退車両のレア演出として設定
- 守り鉄君は通常立ち絵とジャンプ立ち絵を分け、ジャンプ中のみ専用モーションを表示
- ヘルメット正面のロゴ表現は削除し、公開用カード画像もロゴなし版へ差し替え

## 検証項目

- JavaScript構文チェック
- PC操作: キーボード、マウスボタン
- スマートフォン幅: ボタン表示、Canvas比率、横スクロール
- Canvas直接操作: 左右端ホールド、下中央タップ
- 画像読み込みと透過表示
- ゲーム開始、被弾、ゲームオーバー、再挑戦、ハイスコア保存
