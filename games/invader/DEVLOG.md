# Signal Invaders 制作記録

## 企画

生成AIのCodexと対話しながら、PCとスマートフォンの両方で遊べるブラウザゲームを制作する。完成したゲームだけでなく、相談、設計、実装、検証の過程も活動記録として公開する。

## 最初に決めたこと

- ジャンル: シンプルなインベーダーゲーム
- PC操作: 左右キーまたはA・Dキー、スペースキーで射撃
- スマートフォン操作: 画面上の左右ボタン、射撃ボタン
- 基本ルール: 残機3、スコア、ハイスコア、ウェーブ、再挑戦
- 音: Web Audio APIによる効果音と消音切替
- 公開場所: 既存サイト内の `games/invader/`
- 技術: HTML、CSS、JavaScriptのみ。ビルドと外部ゲームライブラリは不要

## 実装上の判断

### Canvasを採用

多数の敵、弾、当たり判定を一定周期で描画するため、ゲーム画面にはCanvasを採用した。敵や自機は画像ファイルではなく矩形の組み合わせで描き、素材の読み込みを不要にした。

### 画面サイズに依存しない座標系

内部解像度を720×820に固定し、CSSで表示サイズを変更する。ゲームロジックは常に同じ座標で計算でき、PCでもスマートフォンでも挙動が変わりにくい。

### タッチ操作

`pointerdown` と `pointerup` を使い、マウスとタッチを共通処理にした。射撃は短いタップでも発射されるよう、押下時に即時実行する。

### データ保存

ハイスコアはブラウザの `localStorage` に保存する。サーバーやユーザー登録を使わず、静的サイトのまま記録を保持できる。

## 初回検証

- JavaScript構文チェック: 成功
- PC表示: Canvas描画、敵編隊、ゲーム開始を確認
- スマートフォン幅: 横スクロールなし、縦横比維持を確認
- タッチ操作: 開始、左右、射撃ボタンの表示と射撃操作を確認
- サイト導線: 活動記録カードからゲームページへのリンクを確認
- ブラウザのJavaScriptエラー: なし
